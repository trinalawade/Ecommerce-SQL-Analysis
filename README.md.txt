 E-Commerce Sales Analysis using SQL


Project Overview

This project focuses on analyzing e-commerce sales data using SQL.
The objective is to extract meaningful business insights related to
customers, products, orders, revenue, and customer behavior.


Business Objectives

- Analyze overall sales performance
- Identify top-performing products
- Find valuable customers
- Analyze category-wise revenue
- Understand payment preferences
- Calculate customer retention
- Generate business KPIs


Database Schema

The project contains 8 relational tables:

1. Customers
2. Categories
3. Products
4. Shippers
5. Orders
6. Order_Items
7. Payments
8. Reviews


Tools Used

- Oracle SQL
- SQL Plus
- Power BI
- Excel


Dataset Size

Customers:
50 records

Products:
30 records

Orders:
200 records

Order Items:
400 records

Payments:
200 records


Key SQL Concepts Used

- SELECT Statements
- Filtering
- Aggregate Functions
- GROUP BY
- HAVING Clause
- Joins
- Subqueries
- Common Table Expressions (CTE)
- Window Functions
- RANK()
- DENSE_RANK()


Business Insights Generated

Revenue Analysis
Calculated total revenue and monthly sales trends.

Customer Analysis
Identified:
- Top customers
- Repeat customers
- Customer lifetime value

Product Analysis
Found:
- Best selling products
- Highest revenue products
- Product rankings

Category Analysis
Analyzed category performance using revenue ranking.
